# 🌿 HTK Farms - E-commerce Web Application

HTK Farms is a full-stack Django-based e-commerce web application designed to sell organic farm products online. The platform allows users to browse products, add items to cart, apply coupons, and place orders with a smooth and user-friendly experience.

---

## 📌 Project Description

This project simulates a real-world online marketplace for farm-fresh organic products. It includes both customer-facing features and an admin panel to manage products, orders, and users.

The system is built with Django following MVC architecture and includes authentication, cart management, order processing, and payment flow (demo-based).

---

## 🚀 Key Features

### 👤 User Features
- User Registration & Login System
- Email OTP Verification
- Browse Products by Categories
- Add to Cart & Manage Cart
- Apply Coupon Codes
- Secure Checkout Process
- Order History & Tracking
- Download Order Receipt (PDF - optional feature)

---

### 🛒 E-commerce Features
- Dynamic Product Listing
- Featured Products Section
- Stock Management
- Price Calculation with Discounts
- Shipping Charges Integration

---

### 🧑‍💼 Admin Features
- Admin Dashboard
- Add / Edit / Delete Products
- Manage Orders & Update Status
- Manage Users
- View Sales Data

---

## 🛠️ Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (Development), PostgreSQL (Production)
- **Deployment:** Render
- **Version Control:** Git & GitHub
